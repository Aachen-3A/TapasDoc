Detailed Description

MUSiC wrapper for crab3.

music_crab is a wrapper script that has been developed to submit the large number of tasks (more or less) automatically, needed to get all data and MC. The script automatically creates CRAB config files according to the dataset file provided. crab3 rebuilds the previous crab2 functionality and relies on the work in the previous version

Tobias Pook
