Detailed Description

Class to collect matplotlib functions.

To use the various matplotlib functions to produce the standard plots (with data[optional], signal[optional], background and uncertainties[optional]). Also different analysis distributions like ratio or siginficance can be added.

Include handling of overflow bins Include cumulative distributions Functionallity for rebinning/variable binning Handling of asymetric errors (systematics) Handling of the data error bars Include file reading functionallity Include hist reweighting

written by Soeren Erdweg 2014-2015

  pos = 0,

Function to add a analysis plot to the figure.

This function is called to add an additional plot to the figure and define its properties, like where it should be placed and how much space of the figure should be taken by this plot. At the moment 'Ratio', 'Diff', 'Signi', 'DiffRatio' and 'SoverSplusB' are available as additional plots.

Parameters
[in]plotString of the plot name that should be added (default = 'Ratio')
[in]posPosition where the plot should be added, 0 is on top of the main plot, 1 and 2 at the bottom (default = 0)
[in]heightHeight of the Plot from the whole plotting range in percent (default = 15)
[in]labelLabel of the y-axis for this additional plot (default = ''[Use the default of this specific analysis plot])
